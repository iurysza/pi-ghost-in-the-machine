# Banner alternatives

All banners are text-free 1792×768 WebP files (exact 21:9). They were generated as independent concepts, center-cropped, tested at 700px and 320px README widths, and optimized below 180KB.

| File | Direction | Notes |
| --- | --- | --- |
| [`terminal-viewport.webp`](terminal-viewport.webp) | Animated face in its terminal habitat | **Canonical.** Best project truth and small-size recognition; includes the cyan face plus restrained thinking, working, done, and error cues. Promoted to [`../banner.webp`](../banner.webp). |
| [`shader-glyph-field.webp`](shader-glyph-field.webp) | ASCII / shader render field | Brightest and most render-native; more abstract, with lifecycle colors reading partly as a spectrum. |
| [`cybernetic-signal-bloom.webp`](cybernetic-signal-bloom.webp) | Original cel-shaded cybernetic cable | Most polished hardware illustration; less specific to Ghostty and the shader glyph language. |
| [`philosophical-leak.webp`](philosophical-leak.webp) | Consciousness leaking through machinery | Strongest mind/body metaphor; intentionally serious and least literal about the project’s tiny face. |

Generated sources are preserved under [`../../ai-artifacts/skills/chatgpt-imagegen/`](../../ai-artifacts/skills/chatgpt-imagegen/) with timestamped names. Generation used `gpt-image-2`, high quality, and a 1792×1024 crop-safe composition; repository assets use a centered 1792×768 crop and WebP quality 82.
